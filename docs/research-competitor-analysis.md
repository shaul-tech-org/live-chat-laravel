---

## 1. 서비스 분류

### A. 메신저 플랫폼 (범용)
- **카카오톡** — 한국 국민 메신저 (B2C 소통 채널)
- **텔레그램** — 글로벌 메신저 (봇 API 활용)

### B. 고객 상담 SaaS (유료)
- **채널톡 (Channel.io)** — 한국 1위 고객 채팅 SaaS
- **센드버드 (Sendbird)** — 글로벌 Chat API/SDK

### C. 오픈소스 고객 채팅
- **Chatwoot** — Ruby/Rails, Intercom 대체
- **Rocket.Chat** — Node.js, 올인원 메시징
- **Tawk.to** — 무료 라이브 챗 (비공개 소스)

### D. 우리 프로젝트
- **LCHAT** — Laravel/PHP, 1줄 위젯, Self-hosted 오픈소스

## 3. 서비스별 장단점

### 카카오톡
| 장점 | 단점 |
|------|------|
| 한국 사용자 100% 설치 | 웹사이트 위젯 불가 |
| 카카오톡 채널 비즈니스 | 해외 사용자 접근 어려움 |
| 알림톡/친구톡 | API 제한적, 실시간 WebSocket 없음 |

### 텔레그램
| 장점 | 단점 |
|------|------|
| 강력한 Bot API | 관리자 대시보드 없음 |
| 완전 무료 | 한국 사용자 비율 낮음 |
| 오픈 프로토콜 | 위젯 형태 미지원 |

### 채널톡 (Channel.io)
| 장점 | 단점 |
|------|------|
| 한국 시장 1위 (B2B SaaS) | 유료 (월 36만원~) |
| AI 상담원 'ALF' | Self-hosted 불가 |
| 마케팅 + CRM 통합 | 데이터 주권 없음 |
| 모바일 SDK 지원 | 커스터마이징 제한 |

### 센드버드 (Sendbird)
| 장점 | 단점 |
|------|------|
| 글로벌 Chat API/SDK | 고가 (MAU 과금) |
| 최고 수준 커스터마이징 | Self-hosted 불가 |
| 영상 통화 지원 | 복잡한 학습 곡선 |

### Chatwoot
| 장점 | 단점 |

## 4. LCHAT 포지셔닝

### 타겟 사용자

```
채널톡/센드버드 → 너무 비싸다
Chatwoot → Ruby라서 커스터마이징 어렵다
Rocket.Chat → 기능이 너무 많고 무겁다
Tawk.to → 오픈소스가 아니다, 데이터 주권이 없다

→ LCHAT: Laravel/PHP 기반, 1줄 설치, 최소 의존성, MIT 오픈소스
```

### 경쟁 우위 5가지

| # | 우위 | 설명 |
|---|------|------|
| 1 | **Laravel 생태계** | PHP 개발자가 바로 커스터마이징 가능. Chatwoot(Ruby)와 Rocket.Chat(Node.js) 대비 PHP 커뮤니티 규모 최대 |
| 2 | **최소 의존성** | PG + MongoDB + Redis 3종만. Chatwoot는 Ruby + Sidekiq + Redis + PG + Node.js 필요 |
| 3 | **1줄 위젯** | `<script>` 태그 하나로 모든 웹사이트에 삽입. Rocket.Chat은 iframe 기반 |
| 4 | **벤치마킹 28기능** | 채널톡/센드버드 수준의 UX. Chatwoot/Tawk.to보다 UX 기능 풍부 |
| 5 | **Built-in Auth** | Keycloak 없이 독립 운영. .env에 이메일/비밀번호만 설정 |

### 기능 커버리지 비교

## 5. 의사결정 요약

### WHY LCHAT?
> "Laravel 개발자를 위한 Self-hosted 라이브 채팅 위젯.
> 채널톡의 UX + Chatwoot의 오픈소스 + Tawk.to의 간편함"

### WHY NOT 기존 솔루션?
| 대안 | WHY NOT |
|------|---------|
| 채널톡 | 유료, Self-hosted 불가, 커스터마이징 제한 |
| 센드버드 | 고가, API만 제공 (위젯 직접 구현 필요) |
| Chatwoot | Ruby/Rails (PHP 커뮤니티 접근 어려움), 설치 복잡 |
| Rocket.Chat | 기능 과다, 위젯 전문이 아님, 무거운 스택 |
| Tawk.to | 비공개 소스, 광고 강제, 데이터 주권 없음 |
